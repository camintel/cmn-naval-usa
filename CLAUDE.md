# Claude Project Notes

## Stack

- React 18, TypeScript, Vite.
- AWS Amplify Gen 2 for hosting, authentication, data, and future backend resources.
- GitHub repository: `camintel/cmn-naval-usa`.

## Commands

- `pnpm install`
- `pnpm dev`
- `pnpm build`
- `pnpm amplify:sandbox`

## Conventions

- Keep pages responsive, accessible, and production-oriented.
- Use `lucide-react` icons for interface controls and status markers.
- Keep generated and local-only files out of git.
- Backend changes should live in `amplify/`.
- Mirror durable workflow guidance between this file and `AGENTS.md`.
